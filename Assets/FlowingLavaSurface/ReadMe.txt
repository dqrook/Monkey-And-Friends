Thankyou for downloading this asset! :)

This package offers 6 kinds of lava surface shaders.
=> Layered Lava
=> Flowing Map
=> Fire Ball
=> Boiling Lava
=> Lava Flow
=> Cartoon Lava
Demo scene demonstrates all these lava effects, please refer it as example usage.

From version 1.4, we support masked lava.
Select Scene and setup "Plane Mask" to "Mask.png", you will see lava surface is masked.
